# Wan2.2-TI2V-5B 風格與參數預設 (新手套用包)

這份是「照抄就能用」的提示詞(prompt)與參數配方。先用最小測試確認能跑,再換成想要的風格。
搭配 ComfyUI 官方「Wan2.2 5B」工作流使用。

---

## 1. 提示詞怎麼寫 (公式)

英文效果通常最好。建議照這個順序組合:

```
[主體] + [動作/表情] + [鏡頭運動] + [場景/環境] + [光線] + [風格/畫質]
```

範例:
```
A young woman with long black hair, smiling and turning her head,
slow cinematic dolly-in, in a sunlit cafe by the window,
warm golden hour light, shallow depth of field, photorealistic, 4k, film grain
```

**鏡頭運動關鍵字**(很有用):
`dolly in / dolly out`(推近/拉遠)、`pan left / pan right`(左右橫搖)、
`tilt up / tilt down`(上下)、`orbit / arc shot`(環繞)、`tracking shot`(跟拍)、
`static shot`(固定鏡頭)、`handheld`(手持感)。

---

## 2. 負向提示詞 (Negative prompt,通用)

直接貼到負向欄位:
```
blurry, low quality, low resolution, distorted, deformed, extra limbs, extra fingers,
bad anatomy, watermark, text, jpeg artifacts, flickering, oversaturated, static, still image
```

---

## 3. 風格預設 (把這段接在主體後面)

| 風格 | 接在提示詞後面的關鍵字 |
|---|---|
| 電影感 Cinematic | `cinematic, dramatic lighting, anamorphic, shallow depth of field, film grain, 35mm` |
| 寫實 Photoreal | `photorealistic, ultra detailed, natural lighting, 4k, realistic skin texture` |
| 動漫 Anime | `anime style, cel shading, vibrant colors, studio anime, clean lineart` |
| 3D 動畫 | `3D render, Pixar style, soft global illumination, subsurface scattering, octane render` |
| 水彩/手繪 | `watercolor painting, hand-drawn, soft brush strokes, pastel colors` |
| 賽博龐克 | `cyberpunk, neon lights, rainy night, reflective wet streets, futuristic city` |

## 4. 參數配方 (12GB VRAM)

> TI2V-5B 原生為 **720p @ 24fps**;官方未針對 480p 調校,低解析度僅供測試。
> 「一次只改一個參數」是調參鐵則。

### 配方 A — 最小測試 (先確認跑得動,最快)
| 參數 | 值 |
|---|---|
| 解析度 | 832 × 480 |
| 長度 frames | 49 (約 2 秒) |
| FPS | 24 |
| Steps | 20 |
| CFG | 5.0 |
| Sampler / Scheduler | euler / simple |
| Shift (ModelSamplingSD3) | 5 |

### 配方 B — 平衡 (日常用)
| 參數 | 值 |
|---|---|
| 解析度 | 1280 × 704 (橫) 或 704 × 1280 (直) |
| 長度 frames | 81 (約 3.4 秒) |
| FPS | 24 |
| Steps | 25 |
| CFG | 5.0 |
| Sampler / Scheduler | uni_pc / simple |
| Shift | 8 |

### 配方 C — 高畫質 (最慢,最吃 VRAM)
| 參數 | 值 |
|---|---|
| 解析度 | 1280 × 704 |
| 長度 frames | 121 (5 秒,官方標準) |
| FPS | 24 |
| Steps | 30 |
| CFG | 5.0 |
| Sampler / Scheduler | uni_pc / simple |
| Shift | 8 |
| 啟動 | 需要時加 `./start.sh --lowvram` |

## 5. 圖生影片 (I2V) 小技巧

- 起始圖放到 `ComfyUI/input/`,在工作流的 Load Image 節點選它。
- 提示詞描述「你希望它怎麼動」,而不是重複描述圖片內容。
  例:`the camera slowly zooms in, the woman blinks and smiles, hair gently moving in the wind`。
- 動作別寫太誇張,5B 在小幅、自然的運動上表現最穩。

---

## 6. 常見問題快速對照

| 想要的效果 | 調哪個 |
|---|---|
| 更聽話、更貼提示詞 | CFG 調高一點 (5 → 6~7);太高會僵硬 |
| 畫面更穩定、少閃爍 | Steps 調高 (25 → 30)、長度縮短 |
| 動作更大 | 提示詞加明確動作 + 鏡頭運動;Shift 試 8 |
| 生成太慢 / 爆顯存 | 降解析度 → 縮短 frames → `--lowvram`;或裝 Sage Attention |
| 畫面太糊 | 提高 Steps、解析度;檢查負向提示詞 |
