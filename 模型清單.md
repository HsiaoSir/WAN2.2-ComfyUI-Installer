# Wan2.2 / Wan2.1 完整模型清單 (ComfyUI 用)

> **75 個檔案全部在 2026-05-29 用 HTTP HEAD 強驗證:回應 200 + Content-Length>0。**
> 每個檔案下方的「旗標」對應 `download_models.sh` 可用的 `--xxx` 參數;沒列旗標的可手動加入。
>
> 來源 repo:
> - [Comfy-Org/Wan_2.2_ComfyUI_Repackaged](https://huggingface.co/Comfy-Org/Wan_2.2_ComfyUI_Repackaged) (`R22`)
> - [Comfy-Org/Wan_2.1_ComfyUI_repackaged](https://huggingface.co/Comfy-Org/Wan_2.1_ComfyUI_repackaged) (`R21`)
>
> URL 一律是 `https://huggingface.co/<REPO>/resolve/main/<REMOTE_PATH>`。
> ComfyUI 自動從 `models/<子資料夾>/` 載入(下載腳本會自動放對位置)。

## 怎麼手動下載 (清單沒有旗標的檔案)

每一筆 URL 都是固定的格式:
```
https://huggingface.co/<REPO>/resolve/main/<REMOTE_PATH>
```

例子(把 `wan2.1_vace_14B_fp16` 抓下來):
```bash
source ~/WAN2.2/ComfyUI/.venv/bin/activate          # 啟用 venv (有 hf CLI)
hf download Comfy-Org/Wan_2.1_ComfyUI_repackaged \
    split_files/diffusion_models/wan2.1_vace_14B_fp16.safetensors \
    --local-dir ~/WAN2.2/ComfyUI/models/diffusion_models/_stage
mv ~/WAN2.2/ComfyUI/models/diffusion_models/_stage/split_files/diffusion_models/wan2.1_vace_14B_fp16.safetensors \
   ~/WAN2.2/ComfyUI/models/diffusion_models/
rm -rf ~/WAN2.2/ComfyUI/models/diffusion_models/_stage
```

或最直接:
```bash
wget -c -O ~/WAN2.2/ComfyUI/models/diffusion_models/wan2.1_vace_14B_fp16.safetensors \
    https://huggingface.co/Comfy-Org/Wan_2.1_ComfyUI_repackaged/resolve/main/split_files/diffusion_models/wan2.1_vace_14B_fp16.safetensors
```
